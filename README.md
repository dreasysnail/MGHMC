# MGHMC
MGHMC (matlab version) implementation of NIPS 2016 paper


***Towards Unifying Hamiltonian Monte Carlo and Slice Sampling***


Demo code on toy cases for MGHMC with different monomial parameter `a`

Run: matlab demo_*.m

# Prerequisite

* Matlab
